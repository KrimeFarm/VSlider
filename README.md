Responsive Slider
=================

This is a fairly stright forward **jQuery 2** plugin, compatible with **Bootstrap** that creates a responsive multi slider animated with **CSS3 transitions**.

## How it works

`
<div class="content">

  <div class="container">
    <div class="slide"><p>content</p></div>
    <div class="slide"><p>content</p></div>
    <div class="slide"><p>content</p></div>
  </div>

  <div class="container">
    <div class="slide"><p>content</p></div>
    <div class="slide"><p>content</p></div>
    <div class="slide"><p>content</p></div>
  </div>

  <div class="container">
    <div class="slide"><p>content</p></div>
    <div class="slide"><p>content</p></div>
    <div class="slide"><p>content</p></div>
  </div>

</div>
`
